# Sistema de Gestion de Tareas Academicas

Aplicacion de escritorio en Java Swing con patron MVC.
Examen practico — Curso de Algoritmos y Estructura de Datos / POO.

## Estructura del proyecto

- `modelo/` -> clases de datos y reglas del sistema
- `vista/` -> interfaz grafica Swing
- `controlador/` -> gestion de eventos y conexion modelo-vista
- `main/` -> punto de entrada de la aplicacion

## Tecnologias

- Java
- Java Swing (sin paleta grafica)
- Patron MVC
